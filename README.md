# Useless Subtitle Player - 无用字幕播放器

> 警告：此项目主要由Ai创作

一个“无聊”却有趣的音频字幕同步播放工具。它会在播放 FLAC 音频的同时，根据 SRT 字幕文件的时间轴，在屏幕的**随机位置**弹出字幕弹窗，让你的字幕体验变得“捉摸不定”。

## ✨ 功能特点

- 🎵 **音频播放**：支持 FLAC 格式音频，支持循环播放与音量调节。
- 📜 **字幕同步**：基于 `.srt` 字幕文件，精准匹配音频时间轴。
- 🪟 **随机弹窗**：每条字幕都会在屏幕工作区内随机位置弹出，避免“呆板”显示。
- ⚙️ **高度可配置**：所有行为均可通过 `config.ini` 自定义（字体、音量、弹窗样式、检查间隔等）。
- 🖥️ **Windows 友好**：自动适配系统 DPI 缩放，弹窗默认置顶。
- 📁 **智能文件查找**：自动识别工作目录下的 `.flac` 和 `.srt` 文件，也支持手动指定路径。

## 🎯 默认资源说明

- **默认字幕文件位置**：  
  `res/Sihan,三Z-STUDIO,HOYO-MiX - DAMIDAMI/` 文件夹下应存放对应的 `.srt` 字幕文件。
- **音频文件**：  
  本程序**不提供** FLAC 音频，请自行获取。  
  音频来源参考：[《绝区零》卢西娅EP | DAMIDAMI](https://www.bilibili.com/video/BV1Ta4gzxE7i/)

> 将下载的 `.flac` 音频文件与 `.srt` 字幕文件放在同一目录（或按上述文件夹结构放置），程序即可自动识别。
