# Useless Subtitle Player - 无用字幕播放器

> 一个故意“无用”却又有点意思的桌面字幕播放器：播放 FLAC 音频，并在屏幕随机位置弹出 SRt 字幕弹窗。

> 警告：该项目由Ai创作

## 简介

**Useless Subtitle Player** 是一个 Python 小工具，它会：

- 播放指定文件夹下的 FLAC 音频文件（或手动指定路径）
- 同步显示同名的 srt 字幕文件
- 每条字幕都会在屏幕**随机位置**弹出一个小弹窗，由用户手动关闭
- 所有行为均可通过 `config.ini` 配置

它没有实用的“自动消失”功能，也不追求优雅的字幕渲染——就是故意“无用”，但胜在简单、可定制，适合用来整蛊、学习或消遣。

## 功能特点

- ✅ 自动查找工作目录下的 `.flac` 和 `.srt` 文件
- ✅ 支持手动指定音频和字幕路径
- ✅ 音频可循环播放 / 单次播放
- ✅ 字幕按时间轴精确弹出
- ✅ 弹窗位置随机（避开任务栏区域）
- ✅ 弹窗样式可配置（字体、边距、按钮文字等）
- ✅ 所有参数均通过 `config.ini` 管理，无需改代码
- ✅ 支持 Windows DPI 缩放感知（弹窗大小适应高分辨率）

## 运行环境

- Python 3.12.*
- Windows（弹窗位置使用了 Windows API，其他系统未测试）
- 依赖库：
  - `pygame`（音频播放）
  - `pysrt`（字幕解析）
  - `tkinter`（Python 自带，用于弹窗）
